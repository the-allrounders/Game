# Game
Hogeschool Rotterdam - MT1F - Project Serious Games - The Allrounders

## Game plot
De dochter van de kikker is ernstig ziek. De enige manier waarop de kikker zijn dochter kan redden is door de magische gouden bloem te vinden. Echter groeit deze bloem alleen op de bodem van de vulkaan, en deze vulkaan staat op uitbarsten! Toch besluit de kikker om het gevaar te trotseren… Eenmaal in de vulkaan vindt de kikker de gouden bloem. Hij kan zijn geluk niet op! Maaaar, dan hoort de kikker gerommel en begint de grond onder zijn poten te schudden. Hij schrikt en probeert zo snel mogelijk weer met de bloem naar boven te klimmen. Hij moet opschieten want de magma komt met een rap tempo omhoog.
